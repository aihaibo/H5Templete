navcollapse 插件只是用来控制导航的展开关闭状态，具体样式需要自己去编写

.nav-collapse 与 .nav-collapse-button 需要在同一个层级

可选参数
{
animate: true, 是否运用动画
speed: 300, 动画时间
fold: true, 窄屏导航条的默认展开状态
navbtn: "-button", 导航开关末尾追加Class
close: "closed", 导航条关闭Class
open: "opend", 导航条打开Class
active: "active" 导航条开关追加Class
}
