本案例运用了 imgload 插件
该插件会用延迟加载的方式加载图片
imgload可选参数
{
threshold : 0, 阀值，正值为距离父级容器 threshold 就开始加载
failure_limit : 0, 容差
event : "scroll", 触发事件，默认滚动条滚动触发
effect : "fadeIn", 图片加载成功后呈现方式
container : window, 图片滚动的父级容器，默认为window
data_attribute : "original", 图片路径存放的属性名，默认为"data-original"
data_bg : "bg", 若有个别图片需要不同的展现方式，可个别设值属性"data-bg"
skip_invisible : false, 跳过隐藏元素中的图片加载
appear : null, 图片加载前触发函数
load : null, 图片加载成功后触发函数
bgsize : "normal", 图片展现方式"normal"、"contain"、"cover"，默认为"normal"
childNode : "i", 图片所加载的容器
trigger_now :false 是否立即出发图片加载
}

.list-02 li a:after 样式中的 padding-bottom:70% 表示高度为宽度的70%，这样可以等比缩放图片
如果需要设定固定高度图片，在 .list-02 li a 中给高度一个定值，则会覆盖 :after 中的高度
